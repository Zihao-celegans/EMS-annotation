# EMS Annotation

A pipeline for analyzing genomic data to identify drug resistance mutations from EMS (Ethyl Methanesulfonate) mutagenesis screens.

## Overview

This repository provides tools for processing and annotating genomic variants from EMS-mutagenized strains to identify candidate drug resistance mutations. EMS is a chemical mutagen that predominantly induces G/C → A/T transitions, and this workflow is tailored to detect and filter such variants in a genomic context.
